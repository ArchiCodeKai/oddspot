Spacing scale · 8 steps × 4 themes

token-based spacing · 4–48px

所有間距（margin / padding / gap）必須走這個 scale，不允許 7px、13px 之類的隨機值。 1–2 (4–8px) 用於元件內部，例如 icon 與文字、badge 內邊距； 3–4 (12–16px) 是元件之間的標準間距； 5–6 (20–24px) 區塊與卡片間； 8–12 (32–48px) 大段落、頁面 padding。 用同一把尺，整個介面就會自動對齊、有節奏感。
