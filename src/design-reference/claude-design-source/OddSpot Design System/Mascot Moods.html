Usage — Copy the SVG + attach the matching .mood-* class to the <svg>. All 6 moods are pure SVG + CSS keyframes (no JS). Drop any one into a React component, or reference this file as a sprite source. Animations use var(--accent) and var(--bg) — they automatically re-theme with the rest of the app.
