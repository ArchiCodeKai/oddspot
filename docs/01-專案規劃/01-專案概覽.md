# 專案概覽

## 定位

**OddSpot** — 台灣城市奇特隱藏景點探索工具

不是旅遊百科（對比 Atlas Obscura），也不是全功能社群平台。
核心體驗：**「附近現在有什麼怪東西值得去一下？」**

## 目標用戶

- 對城市邊角地景感興趣的台灣使用者
- 想填空閒時間做臨時探索的人
- 對廢墟、奇廟、巨物、荒謬景觀有興趣

## 核心體驗

1. 打開 App → 自動定位 → 看到附近奇特景點
2. 地圖模式：地圖上看分布，點擊看簡介
3. 卡片模式：一張一張滑，右滑收藏、左滑跳過
4. 景點詳情：完整資訊 + 傳說 + 推薦時間
5. 一鍵導航至景點

## 差異化

- **台灣在地化**：聚焦台灣，特別是大台北地區
- **景點狀態追蹤**：active / uncertain / disappeared（景點會消失）
- **8 個奇特分類**：怪廟、廢棄、巨物、俗豔、邊緣建築、都市傳說、荒謬景觀、奇店面
- **低門檻探索**：未登入即可瀏覽、收藏（Guest mode）

## 版本規劃

| 版本 | 主要功能 |
|------|----------|
| v1（當前） | 地圖探索、滑卡片、Guest 收藏、登入同步 |
| v2 | 到訪驗證、用戶上傳照片、足跡地圖 |
| v3 | 路線規劃、多點順路 |
