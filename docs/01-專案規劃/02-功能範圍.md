# 功能範圍

| Schema + Seed Data | P0 | v1 | 地基，其他全依賴這個 |

| 景點詳情頁 | P0 | v1 | 地圖點擊後的下一步 |
| 滑卡片探索 | P1 | v1 | 第二種瀏覽方式 |
| Guest mode 收藏 | P1 | v1 | localStorage，不需登入 |
| NextAuth 登入 | P1 | v1 | Google / GitHub OAuth |

| 個人收藏頁 | P2 | v1 | 查看已收藏景點 |
| 足跡地圖 / 熱力圖 | P2 | v2 | 依賴到訪驗證資料 |

## 地圖功能

- 以使用者位置為中心顯示附近景點
- 圓點標記（依分類顏色區分）
- 點擊標記顯示 SpotPopup
- 半徑篩選、分類篩選

## 滑卡片功能

- 向左滑：略過
- 向右滑：收藏
- 向下滑：跳過（不再顯示）
- **不做「向上滑查看更多」**（與 iOS 原生手勢衝突）

## 景點狀態

- `active`：可探索
- `uncertain`：狀況不明（可能已改變）
- `disappeared`：已確認消失

## Guest mode 收藏邏輯

1. 未登入用戶收藏 → 存入 localStorage
2. 用戶登入後 → 自動將 localStorage 收藏 sync 到後端
3. 清空 localStorage 收藏

## 多語言

- 中文（繁體）主要
- 英文次要（nameEn、descriptionEn 欄位）
- 使用 next-intl 管理 i18n
